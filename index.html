Come modificare i giorni festivi nel codice HTML

Apri questo file con un editor di testo (Notepad, VS Code, ecc.) e cerca const HOLIDAYS nel blocco <script>. Troverai:

// Formato chiave: 'YYYY-MM-GG'  valore: 'Nome festività'
const HOLIDAYS = {
  '2026-04-06': 'Lunedì dell\'Angelo',
  '2026-04-25': 'Festa della Liberazione',
  '2026-05-01': 'Festa dei Lavoratori',
  '2026-06-02': 'Festa della Repubblica',
  // '2026-06-24': 'San Giovanni (Torino)',
};

Aggiungere un festivo — inserisci una riga con data 'YYYY-MM-GG' e nome tra apici, seguita da virgola.

Rimuovere un festivo — cancella la riga oppure anteponile // per commentarla.

Nota: i festivi che cadono di sabato o domenica vengono ignorati automaticamente.
